# NeoVim Setup

## Requirement

- [Neovim](https://github.com/neovim/neovim) version 0.5 or higher
- [Nerd Font](https://github.com/ryanoasis/nerd-fonts) Patched fonts for using icon
- [Node.js](https://nodejs.org/)
- [Lua](https://www.lua.org/)

## Installation

git clone https://github.com/pwnrrk/neovim-setup
cd neovim-setup

# Setup language server with npm

sudo npm i -g typescript typescript-language-server eslint emmet-language-server vscode-languageservers-extracted

#For Linux and Mac OS

./install.sh

#For Windows

./install.ps1
